How to Use the Master CAD Library

Grab exactly the 3D printer components you need without bogging down your assemblies.
By Marius

Getting Started with the Library

1
Accessing the Models
Navigating the master document

This master document is designed to be the ultimate reference library for anyone building custom 3D printers, toolheads, or motion systems. Instead of hunting down dozens of obscure datasheets or downloading heavily detailed step files that crash your CAD software, everything here is built as lightweight "clearance bricks."

Viewing the Library

You can access the library directly through your web browser using the public Onshape link provided. You don't need to install any software to view the models, check dimensions, or see how the parts fit together.

ℹ️ Cloud-Based Access

Because the library is hosted on Onshape, any updates, dimensional corrections, or new fan sizes added to the document will be instantly available to you the next time you open the link.

💡 Pro Tip

If you are an Onshape user, you don't need to download anything. Just click "Insert" in your own Assembly, navigate to "Other Documents," paste the library URL, and drop the parts straight into your build.

⚠️ Performance Warning

These models intentionally omit internal geometry like bearing tracks or motor coils. They are strictly for layout, clearance checking, and assembly mapping. Do not try to use these to manufacture replacement internals!

2
Using Configurations
One file, dozens of variations

The real magic of this library is the configuration system. Instead of navigating through folders containing 21 different axial fans or 8 different linear rails, you only deal with a few main "Part Studios."

How to change a part size:

  • Step 1: Open the Part Studio for the component type you need (e.g., NEMA Motors, Axial Fans, MGN Carriages).
  • Step 2: Look at the top-left corner of the viewport for the Configuration Dropdown menu.
  • Step 3: Select your exact spec from the list (like swapping from a 5015 Radial Blower to a 12032). The 3D model, mounting holes, and part names will instantly update.

Final Thoughts

🎉 Time to Build!

If you prefer to work in Fusion360 or SolidWorks, you can easily export these files. Just select the configuration you want, right-click the part in the bottom-left feature tree, and select "Export" to download a clean STEP file.

This library is designed to take the friction out of the design process. Spend less time measuring mounting hole distances, and more time actually building your dream machine. Happy making!
